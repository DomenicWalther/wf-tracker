8 — Accessibility

Focus ring: outline: 2px solid var(--color-accent-light); outline-offset: 2px — applied via :focus-visible on all interactive elements.
All icons are decorative (aria-hidden="true"); text labels always present.
Interactive elements carry aria-label / aria-expanded where text is absent (sidebar collapse button).
Disabled cards use opacity: 0.38; pointer-events: none — not disabled attr — to keep them in the accessibility tree.
Target: WCAG AA contrast minimum throughout; AXE clean.
